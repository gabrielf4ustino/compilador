## Sobre
- Projeto de um compilador implementado em java.

## Pré requisitos
- JDK 18
- Maven 3.8.*

## Instalação
- Java local
  - git clone [url]
  - cd .\compilador\
  - mvn clean install
- Docker
  - git clone [url]
  - cd .\compilador\
  - docker build --rm -t compilador .
## Execução
- mvn exec:java
